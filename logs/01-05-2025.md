## Informasi yang diambil:

| **Nama Variabel**                            | **Deskripsi**                                                |
| -------------------------------------------- | ------------------------------------------------------------ |
| `Name`                                       | Nama lengkap pemain.                                         |
| `Nation`                                     | Negara asal/kebangsaan pemain.                               |
| `Club`                                       | Nama klub tempat pemain bermain.                             |
| `Height`                                     | Tinggi badan pemain (biasanya dalam cm).                     |
| `Weight`                                     | Berat badan pemain (biasanya dalam kg).                      |
| `Age`                                        | Umur pemain (dalam tahun).                                   |
| `Market Value`                               | Nilai pasar pemain (misalnya dalam Euro).                    |
| `Jersey Number`                              | Nomor punggung yang dipakai pemain di klub.                  |
| `Position`                                   | Posisi utama pemain (misalnya: Gelandang, Bek, Penyerang).   |
| `Foot`                                       | Kaki dominan pemain (Kanan/Kiri).                            |
| `Link`                                       | URL ke halaman detail pemain.                                |
| `Pertandingan - Total bermain`               | Total jumlah pertandingan yang pemain ikuti.                 |
| `Pertandingan - Dimulai`                     | Jumlah pertandingan di mana pemain masuk sebagai starter.    |
| `Pertandingan - Menit per tanding`           | Rata-rata menit bermain per pertandingan.                    |
| `Menyerang - Gol`                            | Total gol yang dicetak pemain.                               |
| `Menyerang - Frekuensi gol`                  | Frekuensi rata-rata gol per menit atau pertandingan.         |
| `Menyerang - Tujuan lapangan`                | Jumlah tembakan tepat sasaran.                               |
| `Menyerang - Tembakan per tanding`           | Rata-rata jumlah tembakan per pertandingan.                  |
| `Lewat - Assist`                             | Total assist yang diberikan pemain.                          |
| `Lewat - Lewat`                              | Jumlah total operan/passing.                                 |
| `Lewat - Umpan kunci`                        | Jumlah umpan kunci yang berpotensi menciptakan peluang.      |
| `Lewat - Passing akurat`                     | Persentase akurasi passing pemain.                           |
| `Lewat - Asosiasi lama lulus`                | Total umpan jauh yang sukses.                                |
| `Lewat - Biografi akurat`                    | Total Umpan Silang yang sukses.                              |
| `Bertahan - Intersepsi per tanding`          | Rata-rata intersepsi yang dilakukan per pertandingan.        |
| `Bertahan - Tekel per tanding`               | Rata-rata tekel yang dilakukan per pertandingan.             |
| `Bertahan - Sapuan per tanding`              | Rata-rata sapuan/clearance per pertandingan.                 |
| `Lainnya (rata-rata) - Sukses luar biasa`    | Sukses Dribble                                               |
| `Lainnya (rata-rata) - Pelanggaran`          | Jumlah pelanggaran yang dilakukan pemain per pertandingan.   |
| `Lainnya (rata-rata) - Dilanggar`            | Jumlah pelanggaran yang diterima pemain (dilanggar lawan).   |
| `Lainnya (rata-rata) - Offside`              | Rata-rata offside yang dilakukan pemain per pertandingan.    |
| `Hukuman kartu - Kuning`                     | Jumlah kartu kuning yang diterima pemain.                    |
| `Hukuman kartu - Dua kuning ke merah sebuah` | Jumlah kasus pemain mendapat dua kuning yang berujung merah. |
| `Hukuman kartu - Kartu merah`                | Jumlah kartu merah langsung yang diterima pemain.            |
