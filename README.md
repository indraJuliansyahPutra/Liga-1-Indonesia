# Liga 1 Football Statistics Dashboard

This project aims to create a comprehensive statistical dashboard for Liga 1 football league in Indonesia. The dashboard is built using Power BI and utilizes data scraped from ligaindonesiabaru.com.

## Data Collection

The data for this project is obtained by scraping the ligaindonesiabaru.com website. The information gathered includes player name, club, position, age, nationality, appearances, goals, saves, shots, shots on target, assists, passing, tackles, interceptions, clearances, yellow cards, red cards, fouls, and offsides.

## Dashboard Features

The dashboard provides the following statistical insights:

- Average age of players
- Total goals scored
- Total assists
- Goals distribution by position
- Total yellow cards
- Total red cards
- Total offsides
- Total tackles
- Total player fouls
- Distribution of foreign players by nationality

## Dashboard Visualization

The dashboard is created using Power BI and includes various visualizations such as bar charts, pie charts, and tables to represent the statistical data effectively.

## Files Included

- `data.csv`: CSV file containing the scraped data
- `Liga1_Stats.pbix`: Power BI file containing the dashboard
- `dashboard_screenshot.png`: Screenshot of the dashboard
- `demo_video.mp4`: Video demonstrating the functionality of the dashboard

## Usage

1. Clone this repository to your local machine.
2. Open `Liga1_Stats.pbix` using Power BI Desktop.
3. Refresh the data source to load the latest data if necessary.
4. Explore the dashboard to analyze Liga 1 football statistics.
